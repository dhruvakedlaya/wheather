# ⚡ Weather Forecast

A beautiful and responsive weather forecast application with animated UI and daily email alerts.

🔗 **Live Demo**: [https://dhruvakedlaya.github.io/wheather/](https://dhruvakedlaya.github.io/wheather/)

## Features

- Real-time weather data for any city worldwide
- Current weather conditions with detailed metrics
- 5-day weather forecast
- Geolocation support (My Location button)
- Daily email alert subscription (6:00 AM)
- Sunrise and sunset times
- Beautiful animated dark theme UI

## Weather Details

- Temperature and feels-like temperature
- Wind speed and direction
- Humidity and pressure
- Visibility and cloudiness
- Rain and snow data (when applicable)

## Usage

1. Open `index.html` in your browser
2. Enter a city name or click "MY LOCATION"
3. View current weather and 5-day forecast
4. (Optional) Subscribe to daily email alerts

## Email Alerts

- Enter your email and preferred city
- Receive daily weather updates at 6:00 AM
- Subscriptions are stored locally in your browser

## API

Uses OpenWeatherMap API for weather data.

**Note**: Replace the API key in the code with your own key from [OpenWeatherMap](https://openweathermap.org/api).
